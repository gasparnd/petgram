# Petgram

A mobile photographies app of pets, made with React and GraphQL

## Start

```bash
git@github.com:gasparnd/petgram.git
cd petgram
```

## Installation

```bash
npm install
npm run dev
```

## Thecnologies 
This Progresive Web App project was created with React, GraphQL and Styled Components.
